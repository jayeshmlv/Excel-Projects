# Microsoft Excel Projects

This repository contains Excel projects focused on solving real business problems using data analysis, automation, and reporting techniques.

---

## Projects Included

### 1. Style Advisor Performance Evaluation

Analyzed customer evaluation data to measure Style Advisor performance across different regions using Pivot Tables and charts. Generated business insights and recommendations to improve employee performance and customer satisfaction.

---

### 2. Asset Depreciation Calculator

Developed an automated Excel calculator to estimate asset depreciation using financial formulas. The tool simplifies depreciation calculations and reduces manual effort.
